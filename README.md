# Крестики-нолики (первая версия)

Простая игра «Крестики-нолики» с графическим интерфейсом Windows Forms.

## Начало работы

1. Убедитесь, что установлена среда разработки Visual Studio (любая версия с поддержкой .NET Framework).
2. Откройте файл решения `TicTacToe.sln`.
3. Соберите проект (Build → Build Solution).
4. Запустите приложение (F5).

## Функциональность первой версии

- Игровое поле 3×3.
- Два игрока по очереди ставят X и O.
- Определение победителя или ничьей.
- Кнопка «Сбросить игру» для начала новой партии.
- Возможность вернуться в главное меню.
